# ECNU选课系统 Flutter客户端

## 安装

```bash
flutter pub get
```

## 运行

```bash
flutter run
```

## 构建

### Android
```bash
flutter build apk
```

### iOS
```bash
flutter build ios
```

### Windows
```bash
flutter build windows
```

### macOS
```bash
flutter build macos
```

### Linux
```bash
flutter build linux
```

## 功能

- 浏览器登录获取Cookie
- 浏览和搜索课程
- 选课和退课
- 抢课功能
- 查看已选课程
